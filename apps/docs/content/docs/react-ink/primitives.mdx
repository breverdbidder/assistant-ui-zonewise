---
title: Primitives
description: Composable components for building terminal chat UIs.
---

All primitives are built on [React Ink](https://github.com/vadimdemedes/ink) components and accept standard Ink props.

## Thread

### ThreadRoot

Container for thread content.

```tsx
import { ThreadRoot } from "@assistant-ui/react-ink";
import { Box } from "ink";

<ThreadRoot>
  {/* Thread content */}
</ThreadRoot>
```

### ThreadMessages

Renders messages with a custom component.

```tsx
import { ThreadMessages } from "@assistant-ui/react-ink";

<ThreadMessages
  renderMessage={({ message, index }) => (
    <MyMessage message={message} />
  )}
/>
```

| Prop | Type | Description |
|------|------|-------------|
| `renderMessage` | `(props: { message: ThreadMessage; index: number }) => ReactElement` | Render function for each message |

### ThreadEmpty

Renders children when the thread has no messages.

```tsx
import { ThreadEmpty } from "@assistant-ui/react-ink";
import { Text } from "ink";

<ThreadEmpty>
  <Text dimColor>No messages yet. Start typing!</Text>
</ThreadEmpty>
```

### ThreadIf

Conditional rendering based on thread state.

```tsx
import { ThreadIf } from "@assistant-ui/react-ink";

<ThreadIf running>
  <Text>Generating...</Text>
</ThreadIf>
```

| Prop | Type | Description |
|------|------|-------------|
| `running` | `boolean` | Show when model is running |
| `empty` | `boolean` | Show when thread is empty |

## Message

### MessageContent

Renders message parts with custom renderers.

```tsx
import { MessageContent } from "@assistant-ui/react-ink";
import { Text } from "ink";

<MessageContent
  renderText={({ part }) => <Text>{part.text}</Text>}
  renderToolCall={({ part }) => <Text>[Tool: {part.toolName}]</Text>}
/>
```

| Prop | Type | Description |
|------|------|-------------|
| `renderText` | `(props) => ReactElement` | Custom text renderer |
| `renderToolCall` | `(props) => ReactElement` | Custom tool call renderer |
| `renderImage` | `(props) => ReactElement` | Custom image renderer |
| `renderReasoning` | `(props) => ReactElement` | Custom reasoning renderer |
| `renderSource` | `(props) => ReactElement` | Custom source renderer |
| `renderFile` | `(props) => ReactElement` | Custom file renderer |
| `renderData` | `(props) => ReactElement` | Custom data renderer |

### MessagePrimitiveParts

### MessageIf

Conditional rendering based on message state.

```tsx
import { MessageIf } from "@assistant-ui/react-ink";

  <Text color="green">User message</Text>
</MessageIf>

  <Text color="blue">Assistant message</Text>
</MessageIf>
```

| Prop | Type | Description |
|------|------|-------------|
| `user` | `boolean` | Show for user messages |
| `assistant` | `boolean` | Show for assistant messages |
| `running` | `boolean` | Show when message is generating |
| `last` | `boolean` | Show for last message |

## Tool Call

### ToolFallback

Renders a tool call with status indicators and collapsible details. Used as the default fallback when no custom tool UI is registered.

```tsx
import { ToolFallback } from "@assistant-ui/react-ink";

<ToolFallback
  toolName="search"
  args={{ query: "weather" }}
  argsText='{"query":"weather"}'
/>
```

Auto-expands while running, waiting for approval, or errored; collapses when complete.

#### Status Indicators

| Status | Icon | Color |
|--------|------|-------|
| `running` | Animated spinner | Yellow |
| `complete` | `+` | Green |
| `error` | `x` | Red |
| `requires-action` | `?` | Cyan |

| Prop | Type | Description |
|------|------|-------------|
| `toolName` | `string` | Name of the tool being called |
| `args` | `unknown` | Parsed tool arguments |
| `argsText` | `string` | Raw JSON text of tool arguments |
| `result` | `unknown` | Tool result (if complete) |
| `isError` | `boolean` | Whether the result is an error |
| `status` | `ToolCallMessagePartStatus` | Tool call status from runtime |
| `expanded` | `boolean` | Force expanded/collapsed state |
| `maxArgLines` | `number` | Maximum lines for args when expanded (default: 20) |
| `maxResultLines` | `number` | Maximum lines for result when expanded (default: 20) |
| `maxResultPreviewLines` | `number` | Maximum lines for result preview when collapsed (default: 1) |
| `renderHeader` | `(props) => ReactNode` | Custom header renderer |
| `renderArgs` | `(props) => ReactNode` | Custom args renderer |
| `renderResult` | `(props) => ReactNode` | Custom result renderer |

#### Custom Renderers

```tsx
import { ToolFallback } from "@assistant-ui/react-ink";
import { Text } from "ink";

<ToolFallback
  toolName="search"
  args={{ query: "weather" }}
  argsText='{"query":"weather"}'
  renderHeader={({ toolName, status, expanded }) => (
    <Text bold color={status === "error" ? "red" : "white"}>
      {toolName} ({status})
    </Text>
  )}
  renderArgs={({ args, argsText }) => (
    <Text dimColor>{argsText}</Text>
  )}
  renderResult={({ result, isError }) => (
    <Text color={isError ? "red" : "green"}>
      {JSON.stringify(result)}
    </Text>
  )}
/>
```
